How to use the tool (for mumfali's)

StegMessage takes plain text which you provide and inserts invisible (zero-width) unicode characters to encode the text which you want to hide. The invisible characters can be decoded using the input box given below the plain text and hide message boxes. Once you encode any text, make sure to select everything in the text area, or for convenience, use the copy button given in the top right corner of the box. You can use the copied text in any field which supports UTF-8. The reciever must copy the whole thing and paste it in the decode box to retrieve the hidden text.
